# otel-go-sample
Sample Service for Otel Go 


Step-1
```
docker-compose up
```

Step-2
Access the jaegar UI using:
```
http://localhost:16686/search
```

Step-3
```
go run .
```

Step-4: Hit the curl
```
http://localhost:8080/albums
```

Step-5: Open the jaegar, select the service AsliService and click Find traces
